InterpretStimSynch

Reads stimulus from frame synch file.

Contents

  • Description
  • See Also

Description

synch = InterpretStimSynch(raw_synch, framepts, info) reads the frames framepts of an auxiliary synch file raw_synch, then calculates and returns synch block standard deviation and frequency information in synch.
