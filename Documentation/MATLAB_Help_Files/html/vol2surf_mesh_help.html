vol2surf_mesh

Interpolates volumetric data onto a surface mesh.

Contents

  • Description
  • Visualization Parameters
  • See Also

Description

mesh_out = vol2surf_mesh(mesh_in, volume, dim) takes the mesh mesh_in and interpolates the values of the volumetric data volume at the mesh's surface, using the spatial information in dim. These values are overwritten in a new mesh, which is output as mesh_out.

mesh_out = vol2surf_mesh(mesh_in, volume, dim, params) allows the user to specify parameters for plot creation.

Visualization Parameters

params fields that apply to this function (and their defaults):

NameDefaultEffect
OL0If "overlap" data is presented (OL==1), this sets the interpolation method to "nearest". Default is "linear".
